## Hospital Management System (HMS)

* This project is a simple Hospital Management System implemented in Java (or any general OOP language). It manages core hospital operations such as storing patient records, doctor details, staff details, receptionist and appointments. The system uses Object-Oriented Programming concepts such as classes, objects, inheritance, and polymorphism.

# Features:
Add, view, search, and delete patient records
Add and manage doctor profiles
Schedule and track appointments
Simple, menu-driven interface
Demonstrates:
Encapsulation
Inheritance
Polymorphism
File handling (if included in your version)

# Modules:
Patient Management
Doctor Management
Appointment Scheduling
Reports
Admin Menu

# How to use:
Save the single Java file below as HospitalPatientsAppExtended.java.
Compile: javac HospitalPatientsAppExtended.java
Run: java HospitalPatientsAppExtended
Sample users (created automatically first run):
admin / admin123 (Admin)
doc / doc123 (Doctor)
nurse / nurse123 (Nurse)
